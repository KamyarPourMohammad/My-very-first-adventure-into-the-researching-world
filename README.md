Multi-Label Chest X-ray Classification using DenseNet121
This repository contains a deep learning pipeline for the multi-label classification of thoracic diseases using chest X-ray images. The project is built using PyTorch and leverages the DenseNet121 architecture, specifically optimized for medical imaging tasks.üìå Project OverviewThe goal of this project is to automate the detection of 10 different clinical findings from radiologist reports.
